Launch Aiko locally

            

              Clone the project, install Python dependencies, configure your providers, and start
              the Neural Hub with the Tauri dashboard.
            

          

          

            git clone https://github.com/omax404/Project-Aiko.git
            cd Project-Aiko
            pip install -r requirements.txt
            python start_aiko_tauri.py
