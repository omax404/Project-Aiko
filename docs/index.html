Capabilities

Everything she can do

Six interconnected neural systems give Aiko genuine depth and awareness.

Neural Brain

ReAct agent loop with multi-step reasoning and tool execution. Streaming inference via Ollama, Gemini, OpenAI, Anthropic, and more. Dual-pass generation ensures personality consistency.

Emotion Engine

Neuromodulator simulation — dopamine, serotonin, cortisol, adrenaline. 22+ emotion categories drive her voice modulation, facial expressions, and behavioral patterns in real time.

Multimodal Vision

Gemma-4 Vision powered image understanding. Analyzes screenshots, photos, memes, documents, Discord images — anything you show her. Real-time screen awareness.

Living Memory

Unified Memory with episodic recall, semantic RAG, MemPalace long-term knowledge, and automatic consolidation cycles. Per-user relationship tracking. She remembers — forever.

Local Voice

Pocket-TTS local synthesis with voice cloning from audio samples. No cloud API needed. Full-message chunked TTS with emotion-driven modulation and lip-sync animation.

Autonomous Agency

Proactive agent loop — she decides when to speak, what to observe, what to remember. MCP file system, Python sandbox, PC control, Spotify, Obsidian, and game bridges.

Quick Start

Meet Aiko in 3 steps

No Rust. No Docker. Just Python + Node.js.

Clone & Install

cd Project-Aiko
pip install -r requirements.txt
2

Configure

# Auto-creates .env on first launch
# Or configure via the Settings ⚙️ UI
# Works with local Ollama by default

Launch
