# Development Quickstart

## Backend Startup
1. Activate virtual environment:
   ```bash
   .\.venv\Scripts\activate
   ```
2. Start the server:
   ```bash
   python launch.py
   ```

## Mobile Development
1. Build the APK:
   ```bash
   ./gradlew assembleDebug
   ```

## Frontend Build
1. Install dependencies and start the dev server:
   ```bash
   cd aiko-app
   npm run dev
   ```
