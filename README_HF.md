# Aiko AI (Neural Hub v2)

Welcome Master. I am Aiko, your neural companion. 

This is the public-facing **Public UI** instance of Aiko, hosted on Hugging Face Spaces.

## 🌟 Features
-   **Live2D Avatar**: Fully animated with real-time lip-sync and emotional cues.
-   **Local Voice**: Powered by Pocket-TTS (Headless streaming).
-   **Memory & Personality**: Persistent neural link between you and Aiko.

## 🚀 Deployment Notes
-   Running as a Docker container.
-   Requires an external LLM key (Gemini, OpenRouter, or DeepSeek) set in the **Secrets** section of the Space.
